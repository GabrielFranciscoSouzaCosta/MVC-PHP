Step by Step

    -> 1: 
        criar arquivo da estrutura.html com os elementos que serão utilizados em todas as paginas
    -> 2:
        definir um novo arquivo com conteudo dinamico da pagina: home.html, e adicionar a tag 
        {{conteudo_dinamico}} na estrutura.html
    -> 3:
        -> criar pagina index.php
        -> criar pasta app na raiz do projeto
        -> criar dentro de app o diretorio Template e mover estrutura.html para dentro dela
        -> criar dentro de app o diretorio View e mover para dentro dela o arquivo home.html
        -> pegar o conteudo do Template estrutura.html no arquivo index utilizando a funcao file_get_content
        -> dar eco no conteudo
    -> 4:
        -> Criar pasta Core dentro de App, dentro dela criar o arquivo Core.php e escrever a classe
        -> Instanciar Core na index e chamar metodo de start
    -> 5: 
        -> Criar pasta app/Controller e criar a classe HomeController.php
        -> criar a classe ErroController.php
    -> 6:
        -> pegar o conteudo impresso pelos controllers na index usando ob_start() ob_get_contents() e ob_end_clean()
        -> usar str_replace para alterar o conteudo dinamico e mostrar o template alterado
    -> 7:
        -> criar pasta app/Model
        -> Criar na model a classe Postagem.php
        -> metodos das models serão estaticos para não precisar instanciar os objetos
        -> criar pasta app/lib/Database e dentro criar classe de conexão com padrão singleton
    -> 8:
        -> instalar o composer se ainda não estiver instalado
        -> criar arquivo composer.json na raiz do projeto apenas com o conteudo {"require":{}}
        -> fazer download do composer.phar no site packagist.org, colocar na raiz do projeto
        -> pelo cmd rodar: php composer.phar install
        -> rodar o comando composer require twig/twig (vai criar a pasta vendor e uns trem dentro dela)
        -> no arquivo index.php: require_once 'vendor/autoload.php';
        -> códigos de exemplo do uso do twig estarão disponiveis no arquivo HomeController no metodo index
        -> exemplos de recuperação dos dados nas views.
    -> 9:
        -> adicionar os links dinamicos nas postagens existentes nas views, passando o caminho completo com os parametros get
        -> criar arquivo PostController e a classe
